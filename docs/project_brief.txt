0: Project Brief: AI-Powered Website Audit Tool (MVP)
1: Project Objective
2: Develop a web-based utility that audits any public website URL and generates a detailed audit report based on user-selected audit categories.
3: The objective of the MVP is to create a working proof-of-concept that can analyze websites and generate meaningful reports without incurring any licensing or API costs.
4: Example:
5: Input: https://www.itchotels.com/in/en 
6: Selected Audits:
7: SEO Audit
8: Performance Audit
9: Accessibility Audit
10: Output:
11: Summary Score
12: Detailed Findings
13: Recommendations
14: Downloadable PDF Report
